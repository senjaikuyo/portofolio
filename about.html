Tentang Saya

Lebih dekat dengan SenJai Kuyo: Developer, Mahasiswa, dan Tech Enthusiast.

← Kembali ke Home

Halo! Saya SenJai Kuyo, seorang mahasiswa tingkat akhir program studi Teknik Informatika di Politeknik Negeri Batam (Polibatam). Perjalanan saya di dunia IT dimulai dari rasa penasaran yang mendalam terhadap bagaimana sebuah program dapat mensimulasikan solusi dunia nyata.

Fokus utama saya adalah di bidang Web Development (Full Stack). Saya sangat menikmati merancang arsitektur backend yang solid menggunakan PHP (Laravel) dan Python, namun juga menaruh perhatian besar pada detail antarmuka pengguna (UI/UX) di sisi frontend.

Pendidikan D4 Teknik Informatika, Polibatam (2022-Sekarang)
Lokasi Batam, Kepulauan Riau, Indonesia
Minat Khusus Data Automation, UI/UX Animations, Clean Architecture

Filosofi Coding

Saya percaya bahwa kode bukan sekadar baris instruksi untuk mesin, melainkan komunikasi antara developer dengan timnya (dan diri mereka di masa depan). Oleh karena itu, saya selalu mengutamakan clean code, modularitas, dan pattern yang mudah di-maintain. Saya berpedoman pada prinsip "Buat agar berfungsi, buat agar benar, lalu buat agar cepat."

Di Luar Coding

Saat sedang tidak menatap layar IDE, saya sangat menikmati mendengarkan musik lo-fi, mengeksplorasi desain web eksperimental di Awwwards, atau mempelajari tech stack baru. Saya juga percaya pentingnya berbagi pengetahuan, dibuktikan dari partisipasi saya dalam berbagai pameran PBL (Project Based Learning) di kampus.
